# CodeLab
CodeLab是一个开源数据爬取平台，从GitHub上爬取指定项目的开源数据，包括代码库、issue、Pull request等信息，实现按项目名称、主要语言为关键词的存储和检索，能够实现增量式爬取，并能够完成20个100 星以上的开源项目数据的自动爬取。
目前已完成的页面：
+ 首页
输入项目名称或拥有者用户名，选择主要语言，即可检索出对应的项目。
